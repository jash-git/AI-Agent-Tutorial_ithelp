	Fabric 能夠處理的一些具體任務：
		內容分析：例如 analyze_paper（分析論文）、analyze_prose（分析散文）等。
		內容創作：create_academic_paper（創建學術論文）、create_keynote（創建主題演講）等。
		數據可視化：create_mermaid_visualization（創建 Mermaid 視覺化）、create_markmap_visualization（創建 MarkMap 視覺化）等。
		程式碼相關：explain_code（解釋程式碼）、summarize_git_changes（總結 Git 變更）等。
		安全分析：analyze_malware（分析惡意軟體）、create_security_update（創建安全更新）等。
